Media Queries

Uma Media Query é uma regra CSS que permite aplicar estilos diferentes dependendo do tamanho da tela (ou de outras características do dispositivo). Com ela, você faz o site se adaptar a celulares, tablets e computadores — isso é o que chamamos de design responsivo.

A sintaxe básica é: @media (condição) { /* estilos */ }. O bloco de estilos dentro dela só é aplicado quando a condição for verdadeira. A condição mais comum é max-width (largura máxima da tela) ou min-width (largura mínima da tela).

Exemplo de código CSS:

/* Estilo padrão — vale para qualquer tela */
.caixa-exemplo {
    background-color: #10439F; /* azul */
    color: white;
    padding: 1rem;
    text-align: center;
}

/* Media Query — só vale quando a tela tiver 600px ou menos */
@media (max-width: 600px) {
    .caixa-exemplo {
        background-color: #C65BCF; /* muda para roxo no celular */
        font-size: 0.9rem;
    }
}

Redimensione a janela do navegador para ver a mudança!

Tela larga = azul  |  Tela estreita (≤ 600px) = roxo

Layouts flexíveis (Flexbox)

O Flexbox (Flexible Box Layout) é uma ferramenta que permite alinhar e distribuir espaço entre itens dentro de um contêiner, mesmo quando seus tamanhos são desconhecidos ou dinâmicos.

Propriedades CSS: flex - flex-basis - flex-direction - flex-grow - flex-flow - flex-shrink - flex-wrap - order.

Resumo das Propriedades CSS:

  • flex — atalho para flex-grow, flex-shrink e flex-basis juntos.
  • flex-basis — define o tamanho inicial do item antes de o espaço ser distribuído.
  • flex-direction — define a direção do eixo principal: row (horizontal) ou column (vertical).
  • flex-grow — define o quanto um item pode crescer em relação aos outros.
  • flex-shrink — define o quanto um item pode encolher quando o espaço é insuficiente.
  • flex-flow — atalho para flex-direction + flex-wrap.
  • flex-wrap — controla se os itens quebram para a linha seguinte quando não cabem.
  • order — altera a ordem visual dos itens sem mudar o HTML.

Propriedades de Alinhamento: justify-content - align-items - align-content - align-self - place-content - place-items - row-gap - column-gap - gap.

Resumo das Propriedades de Alinhamento:

  • justify-content — alinha os itens no eixo principal (ex: centralizar horizontalmente).
  • align-items — alinha os itens no eixo cruzado (ex: centralizar verticalmente).
  • align-content — alinha as linhas do flex quando há múltiplas linhas e espaço sobrando.
  • align-self — sobrescreve o align-items para um item individual.
  • place-content — atalho para align-content + justify-content.
  • place-items — atalho para align-items + justify-items.
  • row-gap — define o espaço entre as linhas.
  • column-gap — define o espaço entre as colunas.
  • gap — atalho para row-gap + column-gap.

                        /* Contêiner */
/* Contêiner — desktop */
.container {
    display: flex;          /* ativa o Flexbox */
    flex-direction: row;    /* itens em linha (padrão) */
    flex-wrap: wrap;        /* quebra linha se não couber */
    justify-content: space-between; /* distribui espaço entre os itens */
    align-items: center;    /* centraliza verticalmente */
    gap: 1rem;              /* espaço entre os itens */
}

/* Itens */
.item {
    flex-grow: 1;           /* pode crescer para preencher o espaço */
    flex-shrink: 1;         /* pode encolher se necessário */
    flex-basis: 200px;      /* tamanho inicial de cada item */
    padding: 1rem;
    text-align: center;
}

/* Responsivo — celular (≤ 600px) */
@media (max-width: 600px) {
    .container {
        flex-direction: column; /* empilha os itens verticalmente */
        align-items: stretch;   /* itens ocupam a largura toda */
    }
}
